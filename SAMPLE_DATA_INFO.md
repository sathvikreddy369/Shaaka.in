## üîÑ Resetting Sample Data

If you want to reset the sample data:

```bash
# Remove all data
mysql -u root -p farmer_market -e "
DELETE FROM reviews;
DELETE FROM cart;
DELETE FROM orders;
DELETE FROM food_items;
DELETE FROM crops;
DELETE FROM blogs;
DELETE FROM faqs;
DELETE FROM employees;
DELETE FROM farmers;
DELETE FROM customers;
DELETE FROM admin;
"

# Re-import sample data
mysql -u root -p farmer_market < sample_data.sql
```

Or use the seed script again - it will skip duplicates.
